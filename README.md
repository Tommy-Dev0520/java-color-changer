# Java Color Changer

My first Java Swing project created during my SENA Java course.

## Description

This application allows the user to change the color of a text using three different buttons:

- 🟡 Yellow or "Amarillo"
- 🔵 Blue or "Azul
- 🔴 Red or "Rojo"

This project was created to practice Java Swing components and event handling.

## Technologies

- Java
- Swing
- NetBeans IDE

## What I learned

- Creating graphical interfaces
- Using buttons
- Handling events
- Changing component properties
- Creating my first Java application
