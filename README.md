# Java Color Changer

Mi primer proyecto de Java Swing durante mi curso del SENA.

## Description

Esta aplicacion permite al usuario cambiar el color del texto usando tres botones diferentes:

- 🟡 Yellow or "Amarillo"
- 🔵 Blue or "Azul
- 🔴 Red or "Rojo"

Este proyecto fue creado para practicar componentes de Java Swing y manejo de eventos.

## Technologies I used

- Java
- Swing
- NetBeans IDE

## What I learned?

- Crear interfaces graficas
- Usar botones
- Manejo de eventos
- Cambiar propiedades de texto
